На шахівниці розміром NxN в клітці (x1, y1) стоїть голодний шаховий кінь. Він хоче потрапити до клітини (x2, y2), де росте смачна шахова трава. Яку найменшу кількість ходів він має для цього зробити?

Вхідні дані

Вхідний файл INPUT.TXT містить п'ять чисел: N, x1, y1, x2, y2 (5 < N < 20, 1 < x1, y1, x2, y2 < N). Ліва верхня клітина дошки має координати (1,1), права нижня – (N,N).

Вихідні дані

У вихідний файл OUTPUT.TXT необхідно вивести найменшу кількість ходів коня.

Приклад

№ INPUT.TXT OUTPUT.TXT
1 5 1 1 3 1 2


Будь ласка, завантажте файл з назвою INPUT.TXT.
